## Installation

```bash
$ npm i
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# production build
$ npm run build
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
